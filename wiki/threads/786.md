<p>Thread By: Wild<br /><span style="color:#3366ff;">Wild&#39;s Tutorials!</span><br /><span style="color:#3366ff;">Fix Custom Weapons Not Going Into The Box!</span><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;"><span style="color:#999999;">Hello, do you have the problem where custom weapons that you download and install will not show up in the box? Well I did too, and this is how you fix it!</span></span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Step 1) Make sure you fully install the weapon you downloaded/created. This includes adding all the text it tells you to add, wherever it tells you to add it.</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Step 2) If you are on this step that probably means your weapons still don&#39;t show up in the box. Here&#39;s what you do next.</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">- Go to this location: BO3ROOT\share\raw\gamedata\weapons\zm.</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">- In the zm folder, copy the file named: zm_levelcommon_weapons.csv</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Step 3) Now that you&#39;ve copied that file, head on over to you map&#39;s folder. This can be found by going to: BO3ROOT\usermaps\zm_yourmap or by simply right clicking your map in the launcher and click &quot;Open map folder&quot;</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Next, once inside your map&#39;s folder you want to create a new folder named &quot;gamedata&quot;. Inside of the gamedata folder create another folder named &quot;weapons&quot;. Inside the weapons folder create another folder named &quot;zm&quot;. Inside this zm folder you now want to paste that zm_levelcommon_weapons.csv file we copied from earlier. Now when installing custom weapons you are going to put the text in this location, not the BO3ROOT\share\raw\gamedata\etc. location where it says in the weapon&#39;s instruction.</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Step 4) Open up your map&#39;s GSC file. This should be located here: BO3ROOT\usermaps\scripts\zm_yourmap.gsc. Do the same for your map&#39;s CSC.</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">At the very bottom under everything add this function if it is not already there:</span></p><br />{% highlight php %}
function custom_add_weapons()
{
zm_weapons::load_weapon_spec_from_table("gamedata/weapons/zm/zm_levelcommon_weapons.csv", 1);
}
{% endhighlight %}
 <br /><span style="color:#999999;">It should already be in both the GSC and CSC by default, but you need to make sure just in case.</span><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Step 5) Open up your map&#39;s zone file. Easiest way is to right click your map in the launcher and click &quot;Edit Zone File&quot;.</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Once here you want to find where you added all the text for your weapons that it said to add to the zone file. Under all of it, add this text if it&#39;s not already there:</span></p><br />{% highlight php %}
stringtable,gamedata/weapons/zm/zm_levelcommon_weapons.csv
{% endhighlight %}
 <br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Step 6) Finally, compile and link your map and all that fun stuff and your weapons should now appear in the box! This most likely will also fix any problem you have with wall buys.</span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;"><strong><span style="color:#999999;">&quot;How do I know this will work?&quot;</span></strong></span></p><br /><p style="text-align:left;"></p><br /><p style="text-align:left;"><span style="color:#999999;">Well it worked for me when I had this problem and every one else who had this problem I told them to do the same thing and it worked for them. So with that said, unless your problem is related to some random thing then this should work for you too!</span></p></p>
